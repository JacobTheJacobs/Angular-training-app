
# angular-training-app
## pure angular app made with firebase responsive to mobile screens please view at screenshots
#### app that selects training from firestore has small calculation of calorie burn per time trained and then stores all data to the backend can be view in past exercises

##### to run app need to please install flexlayout and angular material also needs enviorment folder for firebase configuration
